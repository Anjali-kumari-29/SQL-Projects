# 🎬 Netflix Data Analysis (SQL Project)

## 📌 Project Overview
This project analyzes Netflix dataset using SQL to understand content distribution, genre trends, and release patterns.

## 🗂️ Dataset Information
The dataset includes:
- Show ID
- Type (Movie / TV Show)
- Title
- Director
- Country
- Date Added
- Release Year
- Rating
- Duration
- Genre (Listed_in)

## 🛠️ SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- COUNT()
- AVG()

## 📊 Analysis Performed
- Movies vs TV Shows count  
- Most common genre  
- Movies released after 2015  
- Content distribution by country  
- Average release year  

## 📈 Key Insights
- Movies are more than TV Shows  
- Sci-Fi is one of the most common genres  
- Most content is released after 2015  
- Content is produced by multiple countries  

## 💻 Tools Used
- SQL

## ✅ Conclusion
This project demonstrates SQL skills for data analysis, including filtering, grouping, and extracting insights from structured data.

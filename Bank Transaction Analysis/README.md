# 💳 Bank Transaction Analysis (SQL Project)

## 📌 Overview
This project focuses on analyzing banking data using SQL to understand customer transactions, account details, and financial trends.

---

## 📁 Database Structure

### 🧑 Customers
- Customer_ID  
- Customer_Name  
- City  

### 🏦 Accounts
- Account_ID  
- Customer_ID  
- Account_type  
- Balance  

### 💸 Transactions
- transaction_id  
- account_id  
- transaction_type  
- amount  
- transaction_date  

---

## 📊 Analysis Performed

- Retrieved customer transaction details using JOIN operations  
- Calculated total balance for each customer  
- Compared deposit and withdrawal amounts  
- Identified customers with highest balance  
- Analyzed monthly transaction trends  

---

## 🛠️ SQL Concepts Used

- JOIN  
- GROUP BY  
- Aggregate Functions (SUM, COUNT)  
- ORDER BY  
- Date Functions  

---

## 🎯 Key Insights

- Identified high-value customers based on balance  
- Observed transaction patterns (deposit vs withdrawal)  
- Analyzed monthly transaction flow

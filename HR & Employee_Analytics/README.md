# 🏢 HR Analytics and Employee Data Analysis using SQL

## 📌 Overview
This project focuses on analyzing employee data using SQL to understand workforce distribution, salary patterns, and department-wise insights.

---

## 📁 Database Structure

### 👩‍💼 Employees
- emp_id  
- emp_name  
- department_id  
- city  

### 🏢 Departments
- department_id  
- department_name  

### 💰 Salaries
- salary_id  
- emp_id  
- salary  

---

## 📊 Analysis Performed

- Retrieved employee details along with department information  
- Analyzed employee salaries using JOIN operations  
- Calculated average salary for each department  
- Identified highest paid employee  
- Counted number of employees in each department  

---

## 🛠️ SQL Concepts Used

- JOIN  
- GROUP BY  
- Aggregate Functions (AVG, COUNT)  
- ORDER BY  

---

## 🎯 Key Insights

- Identified departments with higher average salaries  
- Observed salary distribution among employees  
- Recognized top-paid employees  
- Analyzed workforce distribution
